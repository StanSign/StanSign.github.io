### if, else

```
if 조건 {
    조건이 만족하면 실행
} else {
    조건이 만족하지 않으면 실행
}
```

**비교 연산자**: ==, !=, >=, <=, >, <

**논리 연산자**: &&, ||, !

### if, else 예시
```
// 다크모드 여부
var isDarkMode: Bool = true

if (isDarkMode == true) {
    print("다크모드 입니다.")
} else {
    print("다크모드가 아닙니다.")
}

if (isDarkMode != true) {
    print("다크모드가 아닙니다.")
}

if isDarkMode == true {
    print("다크모드 입니다.")
}

if isDarkMode {
    print("다크모드 입니다.")
}

if !isDarkMode {
    print("다크모드가 아닙니다.")
}
```

### ?

```
변수1 = 변수2 ? 변수2가 참일 경우 실행 : 변수2가 거짓일 경우 실행
```

### ? 예시

```
var title: String = isDarkMode ? "다크모드 입니다." : "다크모드가 아닙니다."
```
